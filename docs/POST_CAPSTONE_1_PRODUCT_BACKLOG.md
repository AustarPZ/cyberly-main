# Post-Capstone 1 Product Backlog

This backlog records product work that is intentionally not implemented in the Capstone 1 MVP.

## Resources Information Architecture

Future Resources may be split into:

- Malaysia Guides
- External Knowledge Base

### Malaysia Guides

Future properties may include:

- managed through Cyberly Admin;
- multilingual;
- lifecycle controlled;
- Malaysia-focused;
- eligible for CyberGuard grounding where appropriate.

### External Knowledge Base

Future properties may include:

- trusted external organisations;
- official links;
- clear external-source labels;
- different ownership and update responsibility;
- not counted as Cyberly-published guides.

Future counters should distinguish:

- Malaysia Guides;
- External Sources;
- Learning Topics.

Cyberly should not combine those concepts into one Dashboard "Published Guides" count.

## Learning Path Progress

Future Cyberly Learning Path Progress should be implemented only after the product has:

- stable Learning Path Version definitions;
- required/core Scenario sets;
- denominator policy;
- contribution point rules;
- Super Admin governance for contribution limits.

The current MVP must not display a weighted Learning Path percentage.

## Topic Progress Percentages

Future topic-level percentage progress requires:

- stable Topic-to-Scenario mapping;
- required/core Scenario definitions per Topic;
- optional content behavior;
- repeated Scenario policy;
- versioned denominator rules.

Resource completion should remain excluded unless a future feature explicitly records Resource learning activity.

## Super Admin Contribution Configuration

Future Super Admin controls may manage:

- maximum Assessment contribution;
- maximum Scenario contribution;
- maximum Engagement bonus;
- included Core Scenario set;
- contribution model version;
- activation date.

This requires role enforcement, preview before activation, audit history, validation bounds, rollback strategy and clear learner-impact explanation.

## Deferred Features

- Forgot Password remains deferred.
- Malaysia Guides / External Knowledge Base switching is not implemented.
- Resource completion tracking is not implemented.
- Configurable contribution values are not implemented.
- Super Admin controls are not implemented.
