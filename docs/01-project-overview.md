# Project Overview

## Project Title

Cyberly: AI-Powered Interactive Cyber Wellness Toolkit

## Target Users

The primary target users and research population are Malaysian teenagers aged 13-17. Users outside that range may still use the general platform.

## Core Cyber Wellness Purpose

Cyberly is intended to help learners build safer, healthier online habits through guided learning, scenario-based practice, cyber wellness assessment, personalised recommendations, and conversational support.

## Experimental MVP Objective

The MVP should support a controlled research and learning experience that can collect useful interaction and progress data while protecting users and keeping the system explainable, testable, and locally maintainable.

## Core Planned Modules

- User registration and login
- Learner profile
- Cyber wellness assessment
- Adaptive learning and recommendations
- Scenario-based activities
- AI chatbot
- Agentic learning orchestrator
- Progress tracking
- Minimal admin portal
- Experiment data export

## AI And Tooling Roles

- ILMU: primary Malaysian conversational and localisation provider.
- OpenAI: structured reasoning and fallback provider.
- Gemini: optional future multimodal analysis provider.
- Rule-based logic: deterministic scoring, safety checks, eligibility, progress, and recommendation rules.
- Agentic AI: planned orchestrator for guided learner workflows and next-step recommendations.
- Codex: development assistant for implementation, audits, documentation, and project maintenance.
